# deflat_nlp100_2020
NLP100 2020版のdeflat的回答

各種配布データおよび生成物に関しては、権利の都合上含めていません。

該当するチャプターの`data/chapterX/`ディレクトリに、ダウンロードしたそのままのファイル名で配置することで、動くようになります。

|:- | -:|

| python | 3.7.6(Anaconda 2019.03) |
| mecab | 0.996 |
